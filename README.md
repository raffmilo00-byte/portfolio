## Di cosa si tratta

Portfolio one-page per presentarmi a aziende e recruiter come **junior in web development**, con esperienza lavorativa in **Wirlab** (assistenza reti Wi-Fi da remoto) e formazione **Start2Impact**. Il sito comunica chi sono, cosa cerco, competenze, progetti e un modulo di contatto (senza email in chiaro).

**Obiettivo professionale a lungo termine:** AI Engineer, con basi in sviluppo web.

## Contenuto del repository

Questo repository contiene **un solo progetto**: il portfolio. Non ci sono altre cartelle di consegna (cv o pagina web separati): il CV è integrato come `cv.html` e i progetti Demetra e GreenEarth sono in `progetti/`.

```
├── index.html              → ingresso sito (redirect al portfolio)
├── PORTFOLIO.html          → pagina principale del portfolio
├── cv.html                 → curriculum vitae (stampabile)
├── privacy.html            → informativa privacy (form contatti)
├── css/
│   ├── style.css           → stili portfolio
│   └── cv.css              → stili CV
├── js/
│   ├── contact.js          → gestione form
│   └── emailjs-config.js   → configurazione EmailJS (opzionale)
├── progetti/
│   ├── demetra/            → primo progetto HTML/CSS
│   └── greenearth/         → progetto GreenEarth
└── README.md               → questo file
```

## Tecnologie

- HTML5 semantico
- CSS3 (variabili, Flexbox, Grid, media query, tema responsive)
- JavaScript minimo (anno nel footer; form contatti / EmailJS opzionale)

**Nessun framework** (Bootstrap, React, ecc.), come richiesto dall’esercizio.
